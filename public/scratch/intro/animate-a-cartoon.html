Animate a cartoon

Do you watch any cartoons? Which is your favorite? Let's animate some in this project! Click on each character to see their animation.

1) First, use this link to remix the game: Animate a cartoon - Starter. Or add your own characters from your favorite cartoon.

2) Select a character. Make it change costume and say "Hi!" when clicked.

3) Select another character. When it gets clicked, make it grow big, reproduce a sound and then shrink back to its original size.

4) Select another character. When it gets clicked, make it jump in place while reproducing a sound. Hint: To make it jump, move it up and then down the same amount.

5) Select another character. When clicked, make it change the backdrop and say "Let's go!". Make sure you have more than 1 backdrop.

6) Finally, make the last character glide to a random position while reproducing a sound when clicked.
